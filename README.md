# KalenderBeBob

In Google Kalender abonnieren: Google Kalender öffnen. Links auf „+“ neben „Weitere Kalender“ klicken. „Per URL“ auswählen. Die RAW-URL einfügen. „Kalender hinzufügen“ klicken.
https://raw.githubusercontent.com/PhantoSGE/KalenderBeBob/refs/heads/main/BeBobTermine2526.ics

In Outlook abonnieren: Outlook öffnen. In der Kalenderansicht „Kalender hinzufügen → Aus dem Internet“ wählen. RAW-URL einfügen. Bestätigen. In Outlook wird der Kalender ebenfalls automatisch aktualisiert.

In Apple Kalender abonnieren: Apple Kalender öffnen. „Datei → Neues Kalenderabonnement…“ auswählen. RAW-URL einfügen. Aktualisierungsintervall auf stündlich setzen.
